# UtopiaDungeonNotifier

SummonsBoard's utopia schedule notifier

## Usage
```
> bundle exec bin/utopia_dungeon_notifier check ${uid}
```
or
```
> bundle exec bin/utopia_dungeon_notifier schedule ${uid}
```
